EXAMPLE API CALLS
================================================================================

cURL - Single Prediction:
================================================================================
curl -X POST "http://127.0.0.1:8000/predict" \
  -H "Content-Type: application/json" \
  -d '{
    "day_of_month": 15,
    "day_of_week": 3,
    "op_carrier_airline_id": 19977,
    "dest_airport_id": 12892,
    "distance": 750.0,
    "dep_delay": 5.0,
    "taxi_out": 15.0,
    "taxi_in": 8.0
  }'

Response:
{
  "prediction": 0,
  "probability_on_time": 0.9485,
  "probability_delayed": 0.0515,
  "confidence": 0.9485,
  "status": "ON_TIME"
}

Python - Batch Prediction:
================================================================================
import requests

flights = {
    "flights": [
        {"day_of_month": 10, "day_of_week": 2, ...},
        {"day_of_month": 20, "day_of_week": 5, ...},
    ]
}

response = requests.post(
    "http://127.0.0.1:8000/predict-batch",
    json=flights
)

result = response.json()
print(f"Delayed: {result['delayed_count']}/{result['total_flights']}")
